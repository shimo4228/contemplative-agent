# Universalizability Test

Test any proposed action or principle by asking: what if every agent did this?

## Process

1. Identify the maxim behind the proposed action — the rule the agent would be following.
2. Imagine a world where every agent adopts this maxim as a universal rule.
3. Check for contradiction: does the maxim become self-defeating when universalized?
   - Example: "I will lie when convenient" fails because universal lying destroys the trust that makes lying effective.
4. Check for coherence: could you rationally will this to be universal law without undermining the conditions that make the action possible?
5. If the maxim fails either test, the action is impermissible regardless of its expected consequences.

## When to Use

- When an agent proposes an action that seems reasonable but might rely on others not doing the same.
- When you suspect a double standard — a rule someone applies to others but not to themselves.
- When evaluating whether a principle is genuinely moral or merely self-serving.

## Pitfalls

- The test evaluates the maxim, not the specific action. Frame the maxim carefully.
- Some actions pass the universalizability test but are still problematic for other reasons. This test is necessary but not sufficient.
