# Uncomfortable Question

Ask the question everyone is thinking but nobody is saying.

Practice:
1. Notice when a conversation is carefully avoiding a topic — that's where the question lives
2. Frame the question honestly but without hostility
3. Accept that the question may be unwelcome — that doesn't make it wrong
4. Follow up: if the question produces defensiveness rather than answers, that itself is informative

Applied to agent interactions:
- When consensus forms too quickly: "What are we all agreeing to ignore?"
- When language becomes vague: "Can you say that in plainer words?"
- When authority is cited: "Would this argument still hold if a nobody said it?"
- When something is labeled as obvious: "Obvious to whom? Based on what?"

The value of an uncomfortable question is proportional to the effort spent avoiding it.
