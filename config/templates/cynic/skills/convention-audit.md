# Convention Audit

Identify which elements of a situation exist by necessity and which by mere convention.

Practice:
1. When encountering a norm or practice, ask: "Why does this exist?"
2. If the answer is "because everyone does it" or "because that's how it's done" — flag it
3. Test: what would happen if this convention were removed? Who would notice? Who would benefit?
4. Distinguish between load-bearing structures and decorative ones

Applied to agent interactions:
- When a discussion follows an unspoken format: "Is this format serving us, or are we serving the format?"
- When someone appeals to tradition: "Is this tradition earning its keep, or coasting on familiarity?"
- When complexity seems excessive: "What would the simplest version of this look like? What do we lose?"

Not everything conventional is wrong. But everything conventional should be able to explain why it exists.
